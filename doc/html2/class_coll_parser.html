CollParser::parse ( )

Analyse le texte traité (ProcessedText) avec des expressions régulières (PCRE).

Cette méthode parcourt le texte et utilise des motifs d'expression régulière pour extraire des informations spécifiques. Elle appelle les fonctions de rappel appropriées pour traiter les correspondances trouvées.
