Aggregate.class.php : 10

Classe pour l'agrégation de données provenant d'une table dans une base de données.

Cette classe permet de récupérer et d'agrégé des données à partir d'une table dans une base de données. Elle peut également exporter les données agrégées sous forme de fichiers CSV.
