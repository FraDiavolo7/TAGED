Exporte les données fournies au format CSV.

public static exportAsCSV(array<string|int, mixed> $InputData[, string $Separator = ',' ][, string|array<string|int, mixed> $ColumnHeader = self::EXPORT_COLUMN_FROM_DATA ][, string|array<string|int, mixed> $RowHeader = self::EXPORT_ROW_FROM_DATA ][, sring $FilePath = '' ][, array<string|int, mixed> $IgnoreColumns = array() ][, array<string|int, mixed> $IgnoreRows = array() ][, string $ReplaceSep = '' ]) : string

Cette méthode exporte les données fournies au format CSV avec les options spécifiées. Le résultat peut être enregistré dans un fichier si le paramètre $FilePath est fourni. Les libellés des colonnes et des lignes peuvent être gérés avec les paramètres $ColumnHeader et $RowHeader. Les colonnes ou lignes à ignorer dans la présentation peuvent être spécifiées avec $IgnoreColumns et $IgnoreRows. Row Header peut être:

  • EXPORT_ROW_FROM_DATA : Utilise l'index des données d'entrée
  • EXPORT_ROW_ADD_NUM : Un nombre est ajouté artificiellement
  • EXPORT_ROW_NO_HEADER : Rien n'est présenté comme entête
  • un array : Une liste d'index à utiliser (autant d'index que de lignes de données)
  • un string : Le nom de la colonne à présenter comme entête Column Header peut être:
  • EXPORT_COLUMN_FROM_DATA : Utilise l'index des données d'entrée
  • EXPORT_COLUMN_AS_NUM : Utilise un numéro artificiel comme entête
  • EXPORT_COLUMN_NO_HEADER : Rien n'est présenté comme entête
  • an array : Une liste d'index à utiliser (autant d'index que de colonnes de données)
Parameters
$InputData : array<string|int, mixed>

Les données à exporter.

$Separator : string = ','

Le séparateur de champ (facultatif, valeur par défaut : ',').

$ColumnHeader : string|array<string|int, mixed> = self::EXPORT_COLUMN_FROM_DATA

Comment le libellé des colonnes doit être géré.

$RowHeader : string|array<string|int, mixed> = self::EXPORT_ROW_FROM_DATA

Comment le libellé des lignes doit être géré.

$FilePath : sring = ''

Le chemin du fichier dans lequel enregistrer les données CSV (facultatif, valeur par défaut : '').

$IgnoreColumns : array<string|int, mixed> = array()

Liste des colonnes à ignorer dans la présentation (facultatif, valeur par défaut : array()).

$IgnoreRows : array<string|int, mixed> = array()

Liste des lignes à ignorer dans la présentation (facultatif, valeur par défaut : array()).

$ReplaceSep : string = ''
